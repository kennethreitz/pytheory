# Privacy Policy

_Last updated: 2026-06-14_

**composing-with-pytheory** is a Claude Code skill: a set of Markdown
instructions and code examples that Claude reads locally to help you compose
music with [PyTheory](https://pytheory.org).

## What we collect

Nothing. This plugin does not collect, store, transmit, or share any personal
data, usage data, telemetry, or analytics. It has no servers, no accounts, and
no network calls of its own.

## How it works

The skill instructs Claude to write and run PyTheory Python code on your own
machine (or wherever you run Claude Code). Any files it produces — scripts,
WAV/MIDI exports — stay on your system. The PyTheory library itself runs locally
and does not phone home.

## Third parties

Your use of Claude Code is governed by Anthropic's
[Privacy Policy](https://www.anthropic.com/legal/privacy). Installing this
plugin does not change what Claude Code sends to Anthropic. This plugin adds no
third-party services of its own.
